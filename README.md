==========INFORMES ACADÉMICOS EN LATEX==========

====DESCRIPCIÓN====
  Este repositorio contiene una colección de informes y trabajos académicos realizados durante mi formación universitaria. 
  Los documentos están escritos en LaTeX y abarcan diversas áreas como Física (análisis estadístico de mediciones), Simulación de Sistemas, Inteligencia Artificial, Algoritmos y Bases de Datos.
  La estructura del repositorio está diseñada para ser clara, organizada y fácil de navegar, siguiendo buenas prácticas de investigación reproducible.
